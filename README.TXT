===========================================

Instant Weka How-to by Bostjan Kaluza

===========================================

1. About
This folder contains source code described in Instant Weka How-to book by Bostjan Kaluza (http://bostjankaluza.net).

2. Folder contents
 - /dataset - arff dataset used in the book
 - /data - generated arff dataset
 - /lib - weka libraries
 - Test.java - Starting with Java and Weka (Simple)
 - LoadData.java - Loading the data (Simple)
 - FilterAttributes.java - Filtering attributes (Simple)
 - SelectAttributes.java - Selecting attributes (Medium)
 - Classifier.java - Training a classifier (Simple)
 - MyClassifier.java - Building your own classifier (Advanced) 
 - VisualizeTree.java - Tree visualization (Medium) 
 - Evaluate.java - Testing and evaluating your models (Simple)
 - Regression.java - Regression models (Simple) 
 - AssociationRules.java - Association rules (Medium) 
 - Clustering.java - Clustering (Simple) 
 - ReuseModel.java - Re-using models (Medium) 
 - DirectMarketing.java - Data mining in direct marketing (Simple) 
 - StockForecast.java - Using Weka for stock value forecasting (Advanced) 
 - Recommender.java - Recommendation system (Advanced)


3. Install
Please follow the interactions in the first recipe "Starting with Java and Weka (Simple)". All files but StockForecast.java require Weka 3.6.x.
StockForecast.java requires Weka 3.7.7 and Timeseries Forecasting 1.0 module.
